# The Neural Garden - Mental Health Plugin 

The Neural Garden Plugin acts as a extension of Obsidian, to step up the journaling experience.
It adds some features, which should make obsidian a little easier to operate, while also keeping its core functionality unchanged.

The Plugin is packed with supports, helping to increase some mental health aspects & making it easier to reflect, while also offering self-created guidance with the built in support system.

This Plugin exists, because I exist. 
This Plugin exists, because I needed special support in some parts of my life I couldn't find anywhere else.
Having figured out technology can support me with who I am I am more than determined working on what helps me, and maybe even help others along the way
